## Rubric

Use this project rubric to understand and assess the project criteria.

## Technical Implementation

| Criteria | Submission Requirements |
| --- | --- |
| Notebook Execution | The notebook runs top-to-bottom without errors. All cells execute successfully and produce expected outputs such as training logs, metrics, or plots. |
| Dataset Handling | The dataset is loaded correctly and prepared in a form suitable for the chosen deep learning task (image, sequence, or text). |
| Baseline Model Implementation | A deep learning model (CNN, RNN, or Transformer) is correctly implemented in PyTorch, with an architecture appropriate to the task and data. |
| Model Training | The model is trained using an appropriate loss function and optimizer. Training behavior is observable through metrics or logs (for example, loss curves). |
| Experimental Comparison Implementation | A second experimental configuration is implemented that differs from the baseline in exactly one major aspect (architecture, hyperparameter, or training strategy). |
|  |  |
