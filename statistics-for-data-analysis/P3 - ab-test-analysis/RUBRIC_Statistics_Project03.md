## Rubric

Use this project rubric to understand and assess the project criteria.

## Statistical Analyses

| Criteria | Submission Requirements |
| --- | --- |
| Correct Interpretation of Results | The interpretation of all results from different analyses is correct and includes both statistical reasoning and practical reasoning.  - In "Part II - A/B Test", student should correctly interpret the test statistic and p-value. |
| Correct Values for All Statistical Results | All statistical numeric values/results are calculated correctly.  **Tip**: Students can optionally attempt the classroom quizzes to ensure they are calculating the right value in many cases. |
| Appropriate Presentation Slideshow | A slide show is included. Each slide has been update appropriately, as outlined in the template slide show. This includes both updating the results/tables and, where relevant, including an accurate interpretation of those results. |
